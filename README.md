# Online Quiz System

This is a web-based quiz application built using Flask.

## 🚀 Features
- User-friendly interface
- Multiple quiz questions
- Score calculation
- Database using SQLite

## 🛠 Technologies Used
- Python (Flask)
- HTML
- CSS
- Bootstrap
- SQLite

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

## 📁 Project Structure
- app.py
- templates/
- static/
- quiz.db
- requirements.txt
